# agda-cli-hello-world

## Description
A POC for agda programming language.

## Tech stack
- haskell
- python

## Docker stack
- ubuntu:23.10
- rzuckerm/agda

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
